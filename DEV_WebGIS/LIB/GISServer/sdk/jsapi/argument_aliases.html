Preferred Argument Aliases

AMD-compliant modules allow for flexibility in how modules are named once they're loaded. Since AMD does not use global identifiers, it is up to developers to name modules appropriately in their code.

The purpose of this page is to provide a standard naming convention for how to reference Esri ArcGIS API for JavaScript modules in a require callback function. This page is the Esri equivalent for the Dojo AMD modules spreadsheet.

Class: DataSourceProxy
[ AMD Module Require | Legacy Module Require ]
Properties | Methods

dojo.require("esri.opsdashboard.DataSourceProxy");

Description

(Added at v3.14)
DataSourceProxy is a proxy class that represents a operations dashboard data source. Data source instances cannot be directly instantiated. To get a particular data source, extension mixins functions getDataSourceProxy or getDataSourceProxies should be used. The most commonly used function of data source is executeQuery.

Returns the value corresponding to a field name from a feature coming from the data source. Getting the user friendly value can be a complicated series of steps including checking for feature types and/or coded domains.

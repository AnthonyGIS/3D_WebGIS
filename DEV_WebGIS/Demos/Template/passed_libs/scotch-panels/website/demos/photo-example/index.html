Photo Example

Sets and covers a photo over the Scotch Panel's background.

                
                $('.photo-panel').scotchPanel({

                    type: 'image',

                    containerSelector: '.person', // Selects the nearest matching .person container
                    direction: 'top',
                    duration: 300,
                    transition: 'ease',
                    enableEscapeKey: true
                });
